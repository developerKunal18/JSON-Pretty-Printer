# JSON Pretty Printer

A Python script that formats JSON data into a readable structure.

## Description
This project takes a raw JSON string as input and outputs a clean, properly indented version.

## Features
- JSON parsing
- Pretty formatting with indentation
- Error handling for invalid JSON

## Technologies Used
- Python
- json module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python pretty_print.py

4. Enter a JSON string when prompted

## Example

Input:
{"name":"John","age":25}

Output:
{
    "name": "John",
    "age": 25
}
